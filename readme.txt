== Description ==
This plugin provides an easy, lightweight way to let your users sign up for your Newsletter+ list.
You can use it to sign up users for several different lists by placing the widget in the sidebar.
Email validation before user submit the form.
Please note that you will need Newsletter+ Setup on your server before you can use this widget.

= Features: =

* is easy to use
* email validation before submitting to Newsletter+
* encourages the collection of only information that you actually need i.e. an email address (or name if you want)

== Installation ==

To install Newsletter+ Widget, follow these steps:

1.	Download and unzip the plugin
2.	Upload the entire newsletter-widget/ directory to the /wp-content/plugins/ directory
3.	Activate the plugin through the Plugins menu in WordPress
4.  Go to Appearence-> Widget in Admin panel
5. Add you Newsletter+ installation url and Subscription form ID
6. Save it and check in frontend !
